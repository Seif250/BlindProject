# BlindProject (🎓 University Teams Platform )

Connect, Create, Collaborate! A platform for university students to build project teams.

## ✨ Features

- 👤 User profiles & authentication
- 🎯 Create & join project teams
- 🔍 Search available teams
- 💬 WhatsApp integration
- 📱 Responsive design
- 🌐 Arabic RTL support

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- 🎨 Material-UI
- 📡 Axios

### Backend
- 🖥️ Node.js
- 🗄️ MongoDB
- 🔒 JWT Auth

## 🚀 Quick Start

1. Clone & Install:
```bash
git clone https://github.com/Seif250/BlindProject.git
cd BlindProject

# Install dependencies
cd server && npm install
cd ../client && npm install
```

2. Start servers:
```bash
# Backend
cd server && npm start

# Frontend
cd client && npm start
```

Visit http://localhost:3000 🎉
